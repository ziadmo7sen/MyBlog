المقال الأول: خطوات بدء مدونتك على الإنترنت

إن إنشاء مدونة شخصية على الإنترنت أصبح من أبسط الطرق للتعبير عن نفسك ومشاركة معرفتك وأفكارك مع العالم. مع خدمات مثل GitHub Pages، يمكنك إنشاء موقع مجاني تمامًا دون أي تكلفة، والحصول على دومين جاهز مثل username.github.io.

الخطوة الأولى: تجهيز ملفات الموقع

قبل كل شيء، تحتاج إلى إنشاء ملفات HTML الخاصة بموقعك:

  • index.html: الصفحة الرئيسية لموقعك.
  • article1.html: المقال الأول.
  • article2.html … باقي المقالات.

الخطوة الثانية: رفع الملفات على GitHub

لرفع ملفات الموقع على GitHub:

  1. أنشئ Repository جديد باسم موقعك.
  2. اضغط على Add file → Upload files لرفع الملفات.
  3. اضغط Commit changes لتأكيد الرفع.

الخطوة الثالثة: تفعيل GitHub Pages

لتفعيل GitHub Pages:

  • ادخل إلى Settings → Pages.
  • اختر الفرع main أو master.
  • اضغط Save وسيظهر الرابط النهائي لموقعك.

الخطوة الرابعة: التحقق من الموقع

بعد دقائق من التفعيل، يمكنك زيارة موقعك عبر الرابط:

نصائح إضافية

  • اكتب مقالات منتظمة لتزيد من عدد زوار موقعك.
  • استخدم الصور والفيديوهات لجعل المحتوى جذاب.
  • حافظ على تصميم بسيط وسهل التصفح.

باتباع هذه الخطوات، ستكون قد بدأت مدونتك بنجاح ويمكنك إضافة المزيد من المقالات لاحقًا بسهولة.
